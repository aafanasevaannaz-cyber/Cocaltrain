# Cocaltrain

Веб-приложение для тренировки слуха, разучивания песен по фразам и отработки интонации.

## Что внутри

- тренировка по нотам
- тренировка по фразам
- режим полного прохода песни
- микрофон и определение высоты звука
- каталог песен в виде данных, а не захардкоженного кода
- подготовка к добавлению громкости дорожек, транспонирования и темпа

## Стек

- React
- Vite
- TypeScript
- CSS
- Web Audio API

## Ближайший план

1. Базовый экран и выбор режимов
2. Схема песни и загрузка песен из JSON
3. Движок проигрывания эталонных нот
4. Pitch detection с микрофона
5. Тренировка по фразам
6. Громкость минуса и вокала
7. Транспонирование и темп

## Идея архитектуры

Приложение делится на четыре слоя:

1. Данные песни
2. Аудиодвижок
3. Анализ голоса
4. Сценарии тренировки

Это позволяет легко менять песни без переписывания интерфейса.
