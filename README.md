# Cocaltrain

Веб-приложение для тренировки слуха, разучивания песен по фразам и отработки интонации.

## Как запустить локально

1. Установи Node.js 20+ (рекомендуется LTS).
2. Установи зависимости:

   ```bash
   npm install
   ```

3. Запусти dev-сервер:

   ```bash
   npm run dev
   ```

   В dev-режиме включено авто-обновление страницы при изменении файлов в `src/`, `public/` и `index.html`.

4. Открой URL из терминала (обычно `http://localhost:5173`).
5. Для production-сборки:

   ```bash
   npm run build
   npm run preview
   ```

## Что внутри

- тренировка по нотам
- тренировка по фразам
- режим полного прохода песни
- микрофон и определение высоты звука
- каталог песен в виде данных, а не захардкоженного кода
- подготовка к добавлению громкости дорожек, транспонирования и темпа

### С чего начать прямо сейчас

1. Реализовать слой `features/audio`:
   - `AudioEngine` (play/pause/stop, master volume, track volumes).
2. Реализовать слой `features/pitch`:
   - захват микрофона;
   - определение частоты и перевод в MIDI.
3. Связать `audio + pitch` в `features/training`:
   - сравнение целевой ноты с текущей;
   - выдача `in tune / close / off`.
4. Добавить простой экран прогресса:
   - % попаданий по фразам;
   - лучший результат по песне.

## Идея архитектуры

Приложение делится на четыре слоя:

1. Данные песни
2. Аудиодвижок
3. Анализ голоса
4. Сценарии тренировки

Это позволяет легко менять песни без переписывания интерфейса.
